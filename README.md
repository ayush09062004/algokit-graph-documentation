# AlgoKit Graph Documentation

This repository contains the **official documentation, benchmark notebooks, and example projects** for **AlgoKit Graph v1.0.0**.

If the main repository contains the implementation, this repository demonstrates **how to use the library**, **how its algorithms perform**, and **how to build real-world graph applications using AlgoKit Graph**.

> **Main Library:** https://github.com/ayush09062004/algokit-graph

# Testing & Benchmarking

The **tests/** directory contains Jupyter notebooks used for validating and benchmarking every algorithm implemented in AlgoKit Graph.

Each notebook includes

- Correctness verification
- Performance benchmarks
- Stress testing
- Edge case validation
- Scalability analysis
- Usage examples

Current notebooks include

- BFS & DFS
- Connected Components
- Strongly Connected Components
- Disjoint Set Union
- Prim's MST
- Kruskal's MST
- Toy Graph Experiments

These notebooks are intended for developers, contributors, and researchers who want to evaluate the implementation.

---

# Toy Projects

The **toy_project/** directory contains small real-world applications built using AlgoKit Graph.

These projects demonstrate how the library can be used beyond textbook graph problems.

Examples include

- Social Network Analysis
- Recommendation Systems
- Transportation Networks
- Knowledge Graphs
- Dependency Graphs
- Custom Graph Analytics

Each project is self-contained and serves as a practical example of building applications on top of AlgoKit Graph.
